# 传讯 - 双人聊天应用

一个精美的双人聊天应用，具有丰富的交互功能和现代化设计。

## 功能特色

- 🌙 深色/浅色主题切换
- 🎨 多种配色方案
- 💬 双人聊天模拟
- 🃏 塔罗占卜功能
- 🪙 抛硬币互动
- 🎵 音乐播放器
- 💌 幸福信使
- ⚙️ 个性化设置

## 在线体验

点击链接体验在线版本：[部署中...]

## 本地运行

1. 克隆或下载项目
2. 双击 `index.html` 文件
3. 或运行 `python -m http.server 8000` 后访问 `http://localhost:8000`

## 技术栈

- HTML5
- CSS3 (现代样式和动画)
- JavaScript (ES6+)
- Font Awesome 图标
- Google Fonts

## 部署

本项目支持多种部署方式：

1. **GitHub Pages** - 免费静态网站托管
2. **Vercel** - 现代化部署平台
3. **Netlify** - 持续部署平台
4. **Cloud Studio** - 腾讯云部署服务
